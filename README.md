# Sales & Customer Performance Dashboard

This project analyzes business performance from 2015 to 2018 using Power BI.

## Tools Used
- Power BI
- DAX
- Data Analysis

## Key Insights
- West region generates the highest sales
- Technology is the top-performing category
- Sales increased significantly from 2016 to 2018
- Consumer segment contributes over 50% of revenue
- South region shows the lowest performance

## Dashboard Features
- KPI cards (Sales, Orders, Customers, AOV)
- YoY Growth Analysis
- Trend analysis
- Underperforming regions & categories
- Interactive slicers

- Dashboard.pbix – Power BI file
